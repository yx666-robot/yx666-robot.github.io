符号

  • ∅\varnothing∅：空集。
  • ∈\in∈：属于。
  • ∉\notin∈/​：不属于。
  • N\NN：自然数集。
  • N+\N_+N+​ 或 N∗\N^*N∗：正整数集。
  • Z\ZZ：整数集。
  • QQQ：有理数集。
  • R\RR：实数集。

描述法

表示由数字构成的集合：

{x∈A∣P(x)}\{x\in A|P(x)\} {x∈A∣P(x)}

如果 A=RA=\RA=R，或者 P(x)P(x)P(x) 中对 xxx 所在的数域进行了明确，则可以简写成：

{x∣P(x)}\{x|P(x)\} {x∣P(x)}

例子：

{x∈N+∣2≤x≤8}⟺x∈{2,3,4,5,6,7,8}.\{x\in \N_+|2\le x\le8\}\Longleftrightarrow x\in\{2,3,4,5,6,7,8\}. {x∈N+​∣2≤x≤8}⟺x∈{2,3,4,5,6,7,8}.

集合之间关系

  1. AAA 中所有元素都属于 BBB，记作：

    • A⊆B⟺B⊇AA\subseteq B\Longleftrightarrow B\supseteq A A⊆B⟺B⊇A

    • 此时称集合 AAA 包含于集合 BBB，AAA 是 BBB 的子集。

  2. AAA 中所有元素都属于 BBB，且 BBB 至少有一个元素不属于 AAA，记作：

    • A⫋B⟺B⫌AA\subsetneqq B\Longleftrightarrow B\supsetneqq A A⫋B⟺B⫌A

    • 此时称集合 AAA 真包含于集合 BBB，AAA 是 BBB 的真子集。

  3. AAA 中所有元素都属于 BBB，BBB 中所有元素都属于 AAA，记 AAA 和 BBB 的元素相同，记作：

    • A⊆B,B⊆A⟺A=BA\subseteq B,B\subseteq A\Longleftrightarrow A=B A⊆B,B⊆A⟺A=B

    • 此时称集合 AAA 等于 集合 BBB。

  4. 如果 AAA 中有 BBB 中没有的元素，记作：

    • A⊂BA\not\subset B A​⊂B

    • 此时称集合 AAA 不包含于集合 BBB。

充分条件和必要条件

基本定义

  1. 如果“若 ppp，则 qqq”是真命题，即由条件 ppp 可以推出结论 qqq，记作：

    • p⇒qp\Rightarrow q p⇒q

    • 此时称 ppp 是 qqq 的充分条件，qqq 是 ppp 的必要条件。

  2. 如果“若 ppp，则 qqq”是假命题，即由条件 ppp 不能推出结论 qqq，记作：

    • p⇏qp\nRightarrow q p⇏q

    • 此时称 ppp 不是 qqq 的充分条件，qqq 不是 ppp 的必要条件。

充要条件

如果“若 ppp，则 qqq”与它的逆命题“若 qqq，则 ppp”都是真命题。
即既有 p⇒qp\Rightarrow qp⇒q，又有 q⇒pq\Rightarrow pq⇒p 时：

  • p⇔qp\Leftrightarrow q p⇔q

  • 此时称 ppp 是 qqq 的充分必要条件，简称充要条件，也称ppp 与 qqq 等价。

全称量词和存在量词

全称量词

如“全部”“所有”“任意”等这类词语叫做全称量词，用符号 ∀\forall∀ 表示。全称量词命题的一般形式为：

∀x∈M,p(x)\forall x\in M,p(x) ∀x∈M,p(x)

存在量词

如“存在”“至少有一个”等这类词语叫做存在量词，用符号 ∃\exists∃ 表示。全称量词命题的一般形式为：

∃x∈M,p(x)\exists x\in M,p(x) ∃x∈M,p(x)

否定转换

  • ∀x∈M,p(x)\forall x\in M,p(x)∀x∈M,p(x) 的否定：

∃x∈M,¬p(x)\exists x\in M,\lnot p(x) ∃x∈M,¬p(x)

  • ∃x∈M,p(x)\exists x\in M,p(x)∃x∈M,p(x) 的否定：

∀x∈M,¬p(x)\forall x\in M,\lnot p(x) ∀x∈M,¬p(x)
